# 文章发布导入Excel功能需求文档

## 1. 功能概述

在文章发布页面添加"导入Excel"按钮，用户可通过上传Excel文件批量导入文章内容进行发布。

## 2. 功能需求

### 2.1 Excel模板格式

| 字段 | 说明 | 必填 |
|------|------|------|
| 产品 | 关联产品名称 | 是 |
| 网站 | 目标网站名称（如：有目网,三优号,好项目网，多个用逗号分隔） | 是 |
| 标题 | 文章标题 | 是 |
| 内容 | 文章正文内容（段落之间用空行分隔） | 是 |

**内容段落约定：**
- 段落之间用**空行**分隔（即两个换行符 `\n\n`）
- 用户在Excel中填写内容时，每段内容后按一次回车键做空行，再填写下一段
- 系统会自动将空行转换为段落分隔符
- 图片插入位置按段落编号计算（段1、段2、段3...）

### 2.2 内容示例

在Excel中填写内容时：

| 标题 | 内容 |
|------|------|
| 我的文章 | 第一段内容（这里按回车空一行）第二段内容（这里按回车空一行）第三段内容 |

对应解析后的段落结构：
```
段落1：第一段内容
段落2：第二段内容
段落3：第三段内容
```

这样就可以在：
- 段1之前（before_first）插入图片
- 段2之后（段落2, after）插入图片
- 段5之前（段落5, before）插入图片

### 2.3 功能流程

1. 用户点击"导入Excel"按钮
2. 系统弹出文件选择对话框（仅支持.xlsx格式）
3. 用户选择文件并确认上传
4. 系统解析Excel文件，校验数据格式
5. 解析成功后显示预览列表，包含：
   - 文章标题
   - 内容预览（前100字符）
   - 目标网站
   - 关联产品
   - 段落数量
   - 预计发布状态
6. 用户确认预览列表无误后，点击"确认发布"
7. 系统按顺序自动发布每篇文章
8. 发布完成后显示结果汇总（成功/失败数量）

### 2.4 数据校验规则

1. **产品**：必填，必须是系统中已配置的产品
2. **网站**：必填，必须是系统中已配置的目标网站（多个网站用逗号分隔）
3. **标题**：必填，最大200字符
4. **内容**：必填，至少包含1个段落，最大50000字符

### 2.5 错误处理

- Excel格式错误：提示"文件格式不正确，请上传.xlsx文件"
- 数据行为空：跳过该行并记录
- 产品不存在：标记为"产品不存在"，跳过该条
- 网站名称不存在：标记为"网站不存在"，跳过该条
- 发布失败：记录错误信息，继续处理下一条

## 4. 技术实现

### 4.1 前端技术

- 使用Element Plus的Upload组件
- 使用xlsx库解析Excel文件
- 文件上传大小限制：10MB

### 4.2 后端技术

- 使用openpyxl库读取Excel文件
- 复用现有的publish_article函数进行发布
