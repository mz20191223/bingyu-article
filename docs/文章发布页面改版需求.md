# 文章发布页面改版需求文档

## 需求背景

将文章发布页面从当前的"向导式发布"改为"表格批量录入式发布"，让用户可以在一页内批量管理多篇文章的发布，同时保持与Excel导入功能类似的数据录入体验，并支持直接粘贴AI输出的内容。

## 功能需求

### 1. 模式切换

- 提供两个模式选项：「直接录入」和「导入Excel」
- 默认选中「直接录入」模式
- 切换模式时清空当前录入数据（需二次确认）

### 2. 表格录入模式（直接录入）

以表格形式展示，每一行代表一篇待发布的文章：

| 产品 | 目标网站 | AI生成 | 标题 | 内容 | 操作 |
|------|----------|--------|------|------|------|
| [下拉选择] | [多选] | [生成按钮] | [输入框] | [文本域] | [删除] |

#### 字段说明

- **产品**：下拉选择，支持高省、氧惠、麦芽妈妈等
- **目标网站**：下拉多选，支持有目网、三优号、好项目网等
- **AI生成**：点击后调用AI生成内容，自动填充到标题和内容字段
- **标题**：文本输入，单行，不超过100字符，支持AI生成后回填
- **内容**：文本域，支持直接粘贴AI输出的内容（按 `\n\n` 自动分段识别），支持AI生成后回填
- **操作**：删除按钮，点击可删除该行

#### AI生成功能说明

**重要：AI生成时必须根据选择的产品获取对应的配置和数据**

- 用户先选择产品和目标网站
- 点击「AI生成」按钮后，系统根据产品执行以下操作：

  1. **获取AI模板关联**
     - 查找该产品关联的内容模板（ContentPromptTemplate）
     - 查找该产品关联的标题模板（TitlePromptTemplate）
     - 如果产品没有关联模板，使用系统默认模板

  2. **获取产品图片**
     - 查找该产品在「图片管理」中关联的图片
     - 图片按 position_type（before_first/after_last/custom）和 position_value（段落位置）设置插入点
     - 如果没有关联图片，生成纯文字内容

  3. **调用AI生成**
     - 使用产品对应的AI模板调用AI接口
     - 生成标题和内容

  4. **插入图片到内容**
     - 根据图片的 position_type 和 position_value 设置
     - 将图片插入到内容对应的段落位置
     - 图片以 `<p><img src="图片URL" /></p>` 格式插入

- AI生成完成后，自动将标题填入标题字段、内容（含图片）填入内容字段
- 用户可自行修改生成后的标题和内容

#### 图片插入逻辑

```
内容段落结构：
段落1（含小标题）
段落2
段落3
段落4
段落5
段落6（含结尾引导）

图片插入位置：
- before_first：插入到段落1之前
- after_last：插入到最后一段之后
- custom + position_value=3：在段落3之后插入
- before_paragraph + position_value=2：在段落2之前插入
```

#### 表格操作

- **增加行**：点击「+ 添加一行」按钮，在表格末尾新增空白行
- **删除行**：点击该行的「删除」按钮，移除该行数据
- **默认数据**：首次进入时自动生成1行空白数据

### 3. 发布功能

- 表格上方提供「发布全部」按钮
- 点击发布后，遍历所有有效行执行发布
- 每行后面新增「发布进度」列，显示状态和详细描述：

  | 状态 | 图标 | 说明 |
  |------|------|------|
  | 待发布 | ○ | 等待发布 |
  | 发布中... | ◐ | 正在执行，显示当前步骤（如"正在登录..."、"正在填写内容..."、"正在插入图片..."） |
  | ✅ 发布成功 | ✓ | 发布成功 |
  | ❌ 发布失败 | ✗ | 发布失败，显示错误原因（hover可查看详情） |

- 发布进度描述示例：
  - "正在发布到有目网..."
  - "已登录网站，正在填写UEditor..."
  - "正在插入图片..."
  - "有目网发布成功，准备发布到三优号..."
  - "三优号发布失败：登录超时"

- 发布过程中允许用户继续编辑未发布的行
- 发布完成后显示汇总结果（如：成功3篇，失败1篇）

### 4. Excel导入模式

保留现有的导入功能，作为第二种录入方式。

## 数据关联说明

### 产品关联数据

| 数据类型 | 关联方式 | 说明 |
|----------|----------|------|
| AI内容模板 | 产品 → 内容模板 | 每个产品可配置不同的内容生成模板 |
| AI标题模板 | 产品 → 标题模板 | 每个产品可配置不同的标题生成模板 |
| 产品图片 | 产品 → 图片管理 | 每个产品可配置多张图片及插入位置 |

### AI生成流程图

```
用户选择产品 → 获取该产品的AI模板 → 调用AI生成内容 → 获取该产品的图片 → 按规则插入图片 → 回填标题和内容
```

### 发布流程图

```
用户选择产品+目标网站 → 获取该产品的图片 → 按规则将图片插入内容 → 发布到各目标网站
```

## 数据验证

| 字段 | 验证规则 |
|------|----------|
| 产品 | 必填 |
| 目标网站 | 至少选择一项 |
| 标题 | 必填，不超过100字符 |
| 内容 | 必填，至少5个段落（按 `\n\n` 分割） |

## 技术实现

- 前端：Vue 3 + Element Plus
- 表格使用 el-table 组件
- 内容字段支持文本域，按 `\n\n` 自动解析段落
- AI生成功能调用现有后端AI接口，根据产品获取对应模板
- 图片插入逻辑复用 publish_service.py 中的 insert_images 函数
- 发布进度使用状态图标 + 文字描述
- 后端接口复用现有发布API

## 兼容性

- 保留现有的「导入Excel」功能
- 现有AI生成发布流程保持不变
